CRVP

This application is used to predict the car resale value based on the condition and configuration of the car.

Features

This application takes in various vehicle configurations and predict the
price at which the vehicle can be sold. It predicts the price with the accuracy of 86%.

How it works

It uses random forest regression model, which is trained with a huge amount of past
data. The model is deployes in the IBM cloud using Watson Machine learning.
